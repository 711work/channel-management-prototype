# Channel Management Prototype

This repository is a static GitHub Pages prototype.

## Publish With GitHub Pages

1. Create a GitHub repository, for example `channel-prototype`.
2. Push this folder to the repository `main` branch.
3. Open `Settings -> Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Visit:

   `https://YOUR_GITHUB_USERNAME.github.io/channel-prototype/`

The prototype entry file is `index.html`.
